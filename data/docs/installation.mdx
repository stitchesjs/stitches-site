### Install Stitches

Install Stitches from your terminal via npm or yarn.

```bash
# With npm
npm install @stitches/react

# With yarn
yarn add @stitches/react
```

### Import it

Import `styled` from `@stitches/react`.

```js
import { styled } from '@stitches/react';
```

### Use it

Use the `styled` function to create a component and add styles to it.

```jsx line=3-11
import { styled } from '@stitches/react';

const Button = styled('button', {
  backgroundColor: 'gainsboro',
  borderRadius: '9999px',
  fontSize: '13px',
  padding: '10px 15px',
  '&:hover': {
    backgroundColor: 'lightgray',
  },
});
```

### Render it

Finally, render the component.

<Preview>
  <DemoButton />
</Preview>

```jsx line=5
import { styled } from '@stitches/react';

const Button = styled('button', {...});

() => <Button>Button</Button>;
```

### Available functions

Aside from `styled`, these are the other functions available:

```jsx
import {
  styled,
  __css__,
  __globalCss__,
  __keyframes__,
  __theme__,
  __createTheme__,
  __getCssText__,
} from '@stitches/react';
```

Refer to the [API page](/docs/api) to learn more about each of them.

## Configure Stitches

### Create your config file

To configure Stitches, create a `stitches.config.ts` file (`.js` works too) and import the `createStitches` function.

```jsx
// stitches.config.ts
import { createStitches } from '@stitches/react';
```

This function receives a configuration object:

- `theme`: Define your design [theme](/docs/tokens), which map to CSS properties.
- `media`: Define reusable responsive [breakpoints](/docs/breakpoints).
- `utils`: Create custom [utils](/docs/utils) to improve your developer experience.
- `prefix`: Prefix class names and CSS variables to avoid clashes. [Learn more](/docs/api#prefix).
- `themeMap`: Define custom mapping of CSS properties to theme tokens. [Learn more](/docs/api#thememap).

And returns all the [available functions](/docs/installation#available-functions) above.

```js
// stitches.config.ts
import { createStitches } from '@stitches/react';
import type * as Stitches from '@stitches/react';

export const {
  styled,
  css,
  globalCss,
  keyframes,
  getCssText,
  theme,
  createTheme,
  config,
} = createStitches({
  theme: {
    colors: {
      gray400: 'gainsboro',
      gray500: 'lightgray',
    },
  },
  media: {
    bp1: '(min-width: 480px)',
  },
  utils: {
    marginX: (value: Stitches.ScaleValue<'space'>) => ({ marginLeft: value, marginRight: value }),
  },
});
```

> To see all configuration options, refer to the [API page](/docs/api#createStitches).

## Using script tags

If you prefer, you can add Stitches to your site using `script` tags.

```html
<script src="https://unpkg.com/@stitches/react" crossorigin></script>
```

Then use it like this:

```jsx
// without configuration
const { styled, css, globalCss } = stitches;

// with configuration
const { styled } = stitches.createStitches({ ...config });
```

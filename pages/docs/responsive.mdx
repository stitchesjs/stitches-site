title: Responsive
description: 'Learn more about how responsive styles work in Stitches.'
---

To build responsive UI's, you first need to define your breakpoints in the `breakpoints` object:

```jsx
export const { styled, css } = createStyled({
  breakpoints: {
    small: (rule) => rule,
    medium: (rule) => `@media (min-width: 480px) { ${rule} }`,
    large: (rule) => `@media (min-width: 960px) { ${rule} }`,
  },
});
```

Once created, you can use them in the Style Objects:

```jsx
const Layout = styled('div', {
  display: 'flex'
  small: {
    flexDirection: 'column'
  },
  medium: {
    flexDirection: 'row'
  }
});
```

You can also use breakpoints when applying variants:

```jsx live manual
const Button = styled(
  'button',
  {
    appearance: 'none',
    color: 'white',
  },
  {
    variant: {
      primary: {
        backgroundColor: 'red',
      },
      secondary: {
        backgroundColor: 'blue',
      },
    },
  }
);

render(<Button variant={{ small: 'primary', medium: 'secondary' }}>Responsive button</Button>);
```
